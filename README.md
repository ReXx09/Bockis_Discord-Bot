## Bockis Discord Bot

## Features
- Echtzeit-Serviceüberwachung
- Automatische Kanal-Updates
- Historische Statusdaten
- Web-Dashboard
- Prometheus Metriken

## Konfiguration
1. Kopiere `.env.example` zu `.env`
2. Setze Discord Token
3. Starte mit `docker-compose up`

## Endpunkte
- `/health` - Systemstatus
- `/dashboard` - Webinterface
- `/metrics` - Prometheus Daten
